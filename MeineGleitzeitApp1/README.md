# MeineGleitzeit App

Eine einfache Android-App zur Berechnung der Gleitzeit im Arbeitsalltag.

## Funktionen

- Eingabe von Dienstbeginn und Dienstende
- Eingabe der Pausenzeit
- Eingabe der Sollarbeitszeit
- Berechnung der geleisteten Arbeitszeit und des Gleitzeitguthabens/-defizits
- Speicherung der zuletzt eingegebenen Werte

## Voraussetzungen

- Android 5.0 (API Level 21) oder höher

## Installation

Die App ist im Google Play Store erhältlich.
